## 仓库简介
一些数字电路的FPGA实现，基于正点原子新起点FPGA开发板
FPGA芯片为Altera Cyclone IV **EP4CE10F17C8N**
EP4C：Cyclone IV
E(GX)：普通逻辑资源丰富（高速串行收发器）
10：LE逻辑单元约有10K
[C：高速串行收发器数量，没有所以不写]
F(E,Q,U,M)：FBGA（EQFP, PQFP, UBGA, MBGA）
17：引脚数量为256
C(I,A)：民用级0-85℃（工业级-40-100℃，军用级-40-125℃）
8：速度等级，最大工作频率约为400MHz
N(ES)：无铅（工程样片）
